# cp-analyzer

Node + Vite based Codeforces analyzer with premium UI/UX using Tailwind CSS, GSAP animations, Lenis smooth scrolling, and Chart.js visualizations.

## Stack
- Vite
- Tailwind CSS
- Chart.js
- GSAP
- Lenis

## Run locally
1. Install Node.js 18+.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```
4. Build production bundle:
   ```bash
   npm run build
   ```
